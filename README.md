# Meesala Lokesh — Personal Portfolio 🚀

A modern, professional, and fully responsive personal portfolio website built with HTML5, CSS3, and JavaScript. This project showcases my technical skills, projects, certifications, and academic journey as an aspiring Full Stack Developer.

## ✨ Features
- **Dynamic Content Rendering**: Uses a centralized JavaScript data object for easy content management.
- **Glassmorphism UI**: Sleek, modern design with vibrant gradients and subtle animations.
- **Fully Responsive**: Optimized for all devices (Mobile, Tablet, and Desktop).
- **Interactive Project Cards**: Hover effects with clickable links to GitHub and Live Demos.
- **Form Integration**: Fully functional contact form.
- **Custom Design System**: Consistent typography, color palette, and components.

## 🛠️ Tech Stack
- **Structure**: HTML5
- **Style**: Vanilla CSS3 (Custom Design System)
- **Logic**: Vanilla JavaScript (ES6+)
- **Icons**: FontAwesome & Custom SVGs
- **Deployment**: GitHub Pages
